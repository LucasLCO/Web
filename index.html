Proposta

O objetivo do projeto consiste em um sistema que reduza o desperdício de água, realizando um reaproveitamento por meio da captação e filtração com o uso de calhas. Este é apenas um dashboard para vizualização dos dados coletados pelos sensores.
Os objetivos específicos do presente trabalho consistem em:
Realizar a construção de um sistema que age como um filtro.
Realizar a separação dos resíduos da parte líquida.
Filtrar a água e armazená-la. Após a captação, a água é filtrada e armazenada, e seu nível é supervisionado por uma API (Application Programing Interface).
